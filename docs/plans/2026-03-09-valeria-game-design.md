## 0. 핵심 철학

> **"사랑이 승리한다."**

이 게임의 모든 서사는 이 한 문장 위에 세워진다.

| 키워드 | 의미 |
|--------|------|
| **낭만** | 굳이 하면 손해지만, 신념으로 행동하는 것. 합리적 계산보다 사랑과 신념이 앞서는 캐릭터들. |
| **희생** | 자신을 내어주는 행위가 세계를 바꾼다. |
| **구원** | 죄와 거짓 권위로부터의 해방. 물질적·사회적 해방을 넘어 영혼의 해방. |

> 기독교적 세계관을 명시하지 않는다. 다만 사랑, 희생, 섬김, 구원을 게임의 이야기 안에서 실천한다.
> 어두운 스토리도 낭만과 감동으로 풀어낸다.

## 2. 세계관

### 대륙 구성
봉건제가 지배하는 대륙 **발레리아**. 마정석(Aether Crystal)이 에너지원으로 쓰이며 산업혁명이 태동하는 시대.

```
🏰 부유성 (천외천 계급)
   └─ 대륙을 지배한다 자처. 아크론교의 신앙 대상.
      부패하고 허영에 가득 차 있으며, 결국 추락한다.

🌍 지상 (플레이 공간)
   ├─ 아에테르 왕국 (플레이어블 오픈월드)
   ├─ 왕국 A (대화·아이템으로만 암시)
   └─ 왕국 B (대화·아이템으로만 암시)

🚇 지하
   └─ 교통망만 존재 (수도 내 지하철역). 거주 공간 아님.
```

### 사회 구조
- **봉건제**: 왕 → 공작 → 백작 → 남작 → 농민. 땅은 귀족 소유.
- **신흥 자본**: 상인 조합, 광업 회사 등이 돈의 힘으로 귀족에 도전.
- **긴장 구도**: 어느 귀족 영지를 지상 철도가 지나가냐가 정치 싸움이 됨.
- **시대 분위기**: 남보다 잘 살아야 한다는 경쟁적 분위기.

### 운송수단
| 수단 | 용도 | 계층 |
|------|------|------|
| 마차 | 도시 내 이동 | 전 계층 (등급 차이 있음) |
| 지상 철도 | 도시 간 이동 | 중산층 이상 |
| 지하철 | 수도 내 대중교통 | 전 계층 혼재 (계급 충돌 공간) |
| 열기구 비행선 | 장거리·화물·관광 | 부유층, 상인. 지브리풍 디자인. |

> 엔진 기반 프로펠러 비행기는 아직 발명되지 않은 세계.

### 은유 구조
| 게임 요소 | 은유 |
|----------|------|
| 아크론교 | 타락한 제도 종교 |
| 스포라교 | 핍박받지만 대가 없이 이웃을 섬기는 초기 기독교 |
| 부유성의 천외천 | 신이라 자처하는 거짓 권위 |
| 부유성 추락 | 우상의 몰락, 거짓 권위의 붕괴 |
| 스포라교 손 모임 | 희생과 사랑 — 드러나지 않게 세상을 밝힘 |
| 해방의 시대 | 죄와 거짓으로부터의 해방, 진짜 사랑의 시작 |

## 3. 종교

### 아크론교 (Arkron Order) — 국가 공인 주류
- "부유성은 신의 거처, 천외천은 신의 대리인"
- 도시마다 사원. 헌금 강요. 정치와 깊이 결탁.
- 사제들이 의술로 치료 (신의 힘이 아닌 의학).
- 실상은 권력과 이권을 위한 부패 집단.

### 스포라교 (Spora) — 비공인, 핍박받는 소수
- "이웃이 곧 신이다."
- 지하에서 비밀리에 모임. 바라는 것 없이 가난한 자를 무료로 치료.
- 메인 스토리에 직접 개입하지 않고 은은하게 등장.
- 민중이 "손 모임"이라 부르기도 함.

> ⚠️ 이 세계에 신의 힘으로 치유하는 개념은 없음. 모든 치료는 의술.

## 4. 플레이어 캐릭터

- **커스터마이징**: 이름·외모·초기 역할 모두 플레이어가 설정 (완전 커스터마이징)
- **설정**: 발레리아 대륙 아에테르 왕국에 흘러들어온 자. 특정 출신 없음.
- **코-옵**: 각자 자기 캐릭터로 같은 세계를 돌아다님. 스토리 컷씬은 솔로로만 진행.

## 5. 세계 구성 (아에테르 왕국)

오픈월드 하나의 연결된 맵. 5개 권역으로 구분되며, 권역 간 이동 경로에도 몬스터·던전·이벤트 존재.

| 권역 | 성격 | 주요 요소 |
|------|------|----------|
| **수도권** | 정치·상업 중심 | 왕궁, 아크론교 대사원, 지하철역, 용병 길드 |
| **서부 항구권** | 무역·비행선 허브 | 상인 조합 본부, 비행선 정류장, 밀수 루트, 해안 던전 |
| **북부 산악권** | 자원·산업 | 마정석 광산 여러 개, 광업회사 시설, 노동자 착취, 혁명조직 거점 |
| **동부 귀족권** | 봉건제 현장 | 귀족 영지 3~4개, 지상 철도 분쟁 지역, 농민 vs 귀족 갈등 |
| **변경권** | 탐험·던전 | 마법사의 탑, 고대 유적 던전, 숨겨진 스포라 성지, 부유성 추락 예정지 |

## 6. 역할(포지션) 시스템

### 기본 개념
직업을 선택하는 게 아니라, 실제로 사용하는 스킬·무기의 숙련도가 쌓이면 역할이 자동 배정됨.
용병 길드 등록증에 역할이 자동 갱신. NPC 반응, 의뢰 조건, 스킬 해금이 역할에 따라 달라짐.

### 7대 기본 역할
| 역할 | 특징 |
|------|------|
| 검사 | 근접 전투 |
| 마법사 | 마정석·에테르 마법 |
| 엔지니어 | 마정석 기계·포탑 |
| 궁수 | 원거리 사격 |
| 약제사 (Apothecary) | 약 제조·치료·독 |
| 척후 | 정찰·잠입·함정 해제 |
| 전령 | 정보 전달·외교 협상 |

### 숙련도 상승 방식 (OR 조합)
```
숙련도 상승 방법
├── 전투 중 실제 사용 (횟수·기여도)
├── NPC 훈련사에게 교습 (비용 지불)
└── 퀘스트·미션 완료 보상
```
플레이 스타일에 따라 자유롭게 선택. 모든 방법이 동등하게 유효.

### 스킬 트리 구조
- **선형 기반** → 일정 숙련도 이상에서 **2차 전직 분기**
- 중간중간 선택적 스킬(옵션) 배치 → 빌드 다양성 확보
- 2차 전직 후에도 해당 분기 내 선형 진행

```
예시 — 검사 스킬 트리
기본 공격 → 강타 → [선택: 피어싱 샷] → 방어 자세 → [선택: 카운터 스탠스] → 반격기
                                                           ↓ (2차 전직)
                                              ┌────────────────────────┐
                                              │                        │
                                           기사 (방패 특화)        검호 (양손검 특화)
                                   방패 튕기기 → ...        회전참 → [선택: 피의 서약] → ...
```

> ⚠️ 7대 역할 전체 스킬 트리 상세 설계는 후속 기획 항목 (#1 태스크)

## 7. 2차 역할 시스템

- 조직에서 **임명**받는 방식으로 추가 역할 획득.
- 최대 **2개** 조직 역할 보유 가능.
- 확정 조직: **5개** (왕궁, 아크론교, 지하혁명조직, 마법사의 탑, 상인 조합)

### 조직별 역할 및 스킬

#### 🏰 왕궁
| 역할 | 핵심 스킬 | 영향 영역 |
|------|----------|----------|
| **왕실기사** | 왕의 권위로 비전투 제압 선택지 / 귀족 영지 무료 통행 / 왕궁 전용 장비 착용 | 사회·전투 |
| **왕실서기** | 비밀 문서 열람(숨겨진 퀘스트) / 위조 문서 작성(신분 위장) / 원거리 NPC 연락 | 정보·외교 |
| **왕실공병** | 공성 장비 설치·해체 / 철도·지하 인프라 열람(지름길) / 마정석 기계 수리·개조 | 전투·탐험 |

#### ⛪ 아크론교
| 역할 | 핵심 스킬 | 단점 |
|------|----------|------|
| **이단심문관** | 심문 대화(강제 자백) / 아크론 구역 무조건 통행 / 스포라 교인 자동 추적 | 스포라 NPC 영구 적대, 민간인 거부감 |
| **성기사** | 신도 구역 전투 버프 / 교회 시설 무료 이용 | 뇌물·밀수 불가, 진실 폭로 시 정체성 혼란 분기 |
| **아크론 의관** | 치료량 증가·상태이상 해제 / 교회 네트워크로 전국 NPC 정보 / 고급 독·약초 레시피 | — |

#### 🔥 지하혁명조직
| 역할 | 핵심 스킬 | 영향 영역 |
|------|----------|----------|
| **선동가** | NPC 군중 일시 동맹화 / 귀족 NPC 평판 하락 유도 / 비밀 유인물 배포(퀘스트 트리거) | 사회·정치 |
| **암살자** | 숨겨진 제거 의뢰 접근 / 스텔스 강화(발각 범위 감소) / 전용 독 합성 레시피 | 잠입·전투 |
| **혁명투사** | 조직원 임시 동원(전투 지원) / 봉기 구역 자유 이동 / 귀족·교회 시설 침투 보조 | 전투·탐험 |

#### 🔮 마법사의 탑
| 역할 | 핵심 스킬 | 영향 영역 |
|------|----------|----------|
| **마법연구원** | 마법 스킬 슬롯 +1 / 마정석 숨겨진 속성 분석 / 금서 열람(금지 마법 해금) | 전투·강화 |
| **에테르공학자** | 마정석 기계 설계도 접근 / 포탑·기계 강화(엔지니어 시너지) / 스킬 마정석 소비 감소 | 전투·기술 |
| **점성술사** | 퀘스트 결과 미리 힌트 / 시간대별 별자리 버프 / 숨겨진 유물 위치 탐지 | 탐험·정보 |

#### 💰 상인 조합
| 역할 | 핵심 스킬 | 영향 영역 |
|------|----------|----------|
| **중개인** | NPC 매수 대화 선택지 해금 / 적을 돈으로 회유·퇴각 / 비밀 정보 구매(퀘스트 단서) | 외교·정보 |
| **밀수업자** | 금지 아이템 거래 / 봉쇄 구역 우회 루트 해금 / 교회·왕궁 금지 물품 의뢰 접근 | 잠입·탐험 |
| **자본가** | 용병 임시 고용(전투 NPC) / 마정석 시장 조작(적 보급 차단) / 투자로 퀘스트 결과 변경 | 전투·전략 |

### 조합 불가 규칙
| 조합 | 이유 |
|------|------|
| 아크론교 + 지하혁명조직 | 정면 적대 세력 |
| 왕궁 + 지하혁명조직 | 체제 수호 vs 체제 전복 |
| 아크론교 이단심문관 + 상인 조합 밀수업자 | 금지품 단속 vs 금지품 거래 |

### 시너지 조합
| 조합 | 효과 |
|------|------|
| 왕궁 + 마법사의 탑 | 왕실 고급 연구 퀘스트 해금 |
| 왕궁 + 아크론교 | 신정왕권 — 귀족·민간인 최고 순응도. 단, 진실 폭로 시 가장 큰 정체성 위기 분기 |
| 지하혁명조직 + 상인 조합 | 혁명 자금 조달 루트 — 가장 빠른 부유성 침투 경로 |
| 아크론교 + 마법사의 탑 | 금서와 교리 충돌 — 정체성 혼란 스토리 분기 |

## 8. 메인 스토리 뼈대

> *"하늘에 사는 신들은 신이 아니었다"*

1. 플레이어는 아에테르 왕국에 흘러들어온 자로 시작. 용병 길드에 등록.
2. 의뢰를 수행하며 5개 권역을 탐험. 각 조직과 관계를 쌓아감.
3. 아크론교의 모순, 귀족-자본 갈등, 부유성의 실체를 서서히 파악.
4. 클라이맥스: 부유성 침투 시도 → 다른 왕국 외교단 개입.
5. 부유성 추락 → 해방의 시대 개막. 스포라교의 사랑이 세상의 기반이 됨.
